# 專案簡述

## 專案名稱

AI 漫畫短劇製作

## 核心目標

建立一套可複製的 AI 製作流程，快速完成短篇漫畫動畫，並讓角色與視覺風格能穩定延續到下一集。

## 內容定位

- 類型：漫畫感短劇 / 微劇情動畫
- 長度：每集 2 到 3 分鐘
- 形式：角色對話、旁白、轉場與簡單運鏡
- 節奏：短、快、明確衝突

## 製作原則

- 少做複雜連續動畫，多用靜態圖加剪輯節奏
- 固定角色設定，優先維持臉部與服裝一致性
- 每段先獨立完成，再統一剪接
- 每一集都要保留可重用素材

## 主要工具鏈

- ChatGPT：腳本、分鏡、提示詞
- ComfyUI + SDXL：角色圖、場景圖、封面圖
- AnimateDiff：短動畫片段
- ElevenLabs：旁白與角色配音
- Suno：主題音樂或配樂
- DaVinci Resolve：後期剪輯與整合

## 交付內容

- 完整短劇影片
- 封面圖
- 腳本檔
- 分鏡檔
- 配音與音樂素材
- 可延續使用的角色參考圖
